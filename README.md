# Asimo

Asimo is a collection of [Terraform][1] scripts that I use to manage Cloud Infrastructure.

## Prerequisites

- [aws-cli][3]
- [Configure aws-cli][4]
- [Terraform][2]

## Installation

`terraform init`

[2]: https://learn.hashicorp.com/tutorials/terraform/install-cli
[3]: https://aws.amazon.com/cli/
